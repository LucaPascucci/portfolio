Typescritp

Linguaggio è un linguaggio di programmazione open source sviluppato da Microsoft. Estende la sintassi di Javascript ed è destinato a essere compilato in Javascript per poter essere interpretato da qualunque WebBroswer o App.

Caratteristiche

Aggiunge o rende più flessibile e potenti le caratteristiche:
  • Firma dei metodi
  • Classi
  • Interfacce: non vengono compilate quindi sono utili solo in fase di sviluppo
  • Moduli
  • Tipi di dato
  • Tipo Enum
