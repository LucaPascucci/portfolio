Logic Programming

"Rispondi a una domanda cercando la soluzione"
I programmi consistono di dichiarazioni logiche, e l’esecuzione corrisponde alla ricerca sistematica delle dimostrazioni (facts) per tali dichiarazioni facendo uso di regole di inferenza. Particolarmente efficace per i problemi che consistono nell’estrazione di conoscenza da basic facts e relazioni (problemi di Intelligenza Artificiale - AI), ma può anche essere impiegato come paradigma general purpose (Kowalsi paper).

Modello computazionale di riferimento: LOGICA FIRST ORDER

Algoritmi = Logica + Controllo: i programmi devono specificare solo la parte logica, il controllo è gestito dalla macchina astratta.
